Search Tips

  • TERM: Simple keyword search.
  • TERM1 TERM2: Files with all of the given terms.
  • TERM1 or TERM2: Files with any of the given terms.
  • date:2015: Files uploaded in the given year.
  • date:Oct 2015, date:Oct 19 2015, etc: Files uploaded in the given date. The date is parsed flexibly.
  • name:filename: Files with the given exact filename.
  • tuid:ID: Files tagged with the given TUID (IFDB ID string).

This form searches files stored on the IF Archive. If you want to search for IF games by bibliographic information (author, year, platform, competitions or awards) we recommend you search on IFDB instead. This form is better for finding interpreters, development tools, articles, information, and specific files.

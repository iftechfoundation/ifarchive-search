Search Examples

  • zork — Simple keyword search of the Archive description text.
  • "zork iii" — Files with the given phrase.
  • zork mit — Files with all of the given terms.
  • zork OR mit — Files with any of the given terms. (The OR must be capitalized.)
  • zork NOT mit — Files with one term but excluding another.
  • date:2015 — Files uploaded in the given year. (No space after the colon.)
  • date:Oct 2015 — Files uploaded in the given date. The date is parsed flexibly: Oct 10 2015, 20151011, a few other forms.
  • name:wumpus.bas — Files with the given exact filename.
  • tuid:9p8kh3im2j9h2881 — Files tagged with the given TUID (IFDB ID string).

This form searches files stored on the IF Archive. If you want to search for IF games by bibliographic information (author, year, platform, competitions or awards) we recommend you search on IFDB instead. This form is better for finding interpreters, development tools, articles, information, and specific files.
